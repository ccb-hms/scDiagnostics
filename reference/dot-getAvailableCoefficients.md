# Determine Available Coefficient Types

Helper function to determine which coefficient types are available in a
regressPCObject based on the model type and data structure.

## Usage

``` r
.getAvailableCoefficients(x)
```

## Arguments

- x:

  An object of class `regressPCObject` containing regression results
  from `regressPC` function.

## Value

A character vector of available coefficient types.
