# Number of Elements

This function returns the number of elements in a matrix or vector.

## Usage

``` r
nElements(X)
```

## Arguments

- X:

  A matrix or vector.

## Value

An integer representing the number of elements in `X`.

## Details

If `X` is a matrix, the function returns the product of its dimensions.
If `X` is a vector, the function returns its length.
