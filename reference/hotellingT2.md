# Calculate Hotelling's T^2 Statistic

Calculates the Hotelling's T^2 statistic for comparing means of
multivariate data.

## Usage

``` r
hotellingT2(sample1, sample2)
```

## Arguments

- sample1:

  A numeric matrix or data frame of multivariate observations for sample
  1, where rows are observations and columns are variables.

- sample2:

  A numeric matrix or data frame of multivariate observations for sample
  2, with the same structure as sample 1.

## Value

The Hotelling's T^2 statistic.
